# Etch A Sketch

A browser-based Etch A Sketch drawing app built with HTML, CSS, and JavaScript as part of **The Odin Project Foundations** curriculum.

## About

This project recreates a simple Etch A Sketch experience in the browser using JavaScript and DOM manipulation. Users can draw by hovering over a grid of squares and dynamically resize the drawing grid while maintaining the same overall drawing area.

## Features

- Default 16 × 16 drawing grid
- Squares generated dynamically with JavaScript
- Drawing by hovering over grid cells
- Progressive darkening effect (10% darker on each hover until fully black)
- Custom grid size (up to 100 × 100)
- Responsive Flexbox layout
- Fixed 960 × 960 drawing area

## Built With

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (ES6)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/etch-a-sketch.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Click **Change Grid** to choose a new grid size (maximum 100).

5. Move your mouse over the grid to start drawing.
